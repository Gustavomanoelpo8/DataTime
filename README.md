# DataTime

Repositório com exemplos e práticas utilizando manipulação de datas e horários em Python.

## Objetivo

Explorar e entender o uso de datas e horas na programação, incluindo criação, formatação, manipulação e operações com valores temporais.

## Conteúdo

- Criação de datas e horários  
- Formatação de datas  
- Conversão entre formatos  
- Operações com datas (adição e subtração de tempo)  
- Uso de bibliotecas para manipulação temporal
